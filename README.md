# Maison Dorée Boutique ✨

Maison Dorée Boutique is a modern and elegant fashion boutique website built using HTML, CSS, and JavaScript. The website is designed to showcase stylish collections, highlight featured products, and provide visitors with a seamless shopping-inspired browsing experience.

## Overview

This project focuses on creating a luxurious and visually appealing online presence for a fashion boutique. The design emphasizes elegance, simplicity, and responsiveness to ensure an engaging experience across all devices.

## Features

- Elegant and modern design
- Fully responsive layout
- Featured collections showcase
- Product display sections
- Smooth and intuitive navigation
- Mobile-friendly experience
- Clean and organized interface
- Fast loading performance

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Live Demo

🔗 https://maisondoreeboutique.netlify.app/

## Purpose

The purpose of this project is to create a professional boutique website that combines aesthetics and functionality while showcasing front-end web development skills.
